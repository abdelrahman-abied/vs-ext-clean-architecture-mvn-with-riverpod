## [1.0.0] - 2024-12-XX

### Added
- Initial release of Clean Architecture and MVN with RiverPod extension
- Automatic folder structure generation for Flutter projects
- Feature-based architecture with domain, data, and presentation layers
- Automatic file generation with proper imports and templates
- Support for Clean Architecture patterns with MVN (Model-View-Notifier)
- Riverpod integration with notifiers and states
